### 로컬 저장소 초기화
 * git 명령어 실행하기
  ![Alt text](images/1.jpg)
  
     $ git init

 * git 사용자 이메일 설정

     $ git config --global user.email "자신의 이메일"
 * git 사용자 이메일 확인

     $ git config user.email
